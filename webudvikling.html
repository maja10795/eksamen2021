HTML

HTML er et kodesprog, som bliver brugt til at skabe en hjemmeside. HTML er med til at vælge indholdet på siden og strukturen på siden. Fx kan man via forskellige html koder indsætte links, billeder lister mm.
Syntaks Syntaks er hvordan du skriver din HTML kode. Det er en slags ”grammatik” for kodesprog, da det er med til at bestemme hvad der fungere bedst på siden. En HTML-fil startes sådan her:
!DOCTYPE html
Denne kode definerer at filen, som man skal til at arbejde i, er en HTML fil. Dermed bed browseren også hvilken version dokumentet er udarbejdet i.
Head
Denne kode indeholder sidens meta-data.
Body
Denne kode indeholder tekst, billeder, lister mm. Det er indholdet, som brugeren af hjemmesiden kommer til at se.
- Niels Østergaard "intro til html.pdf"

SEO
udgangspunkt i en tøjforretning. Hvis webudvikleren ikke har taget højde for SEO, vil hjemmesiden højest sandsynligt ikke blive vist på nogen søgemaskiner og der vil derfor ikke komme nogen besøgende på siden. Det bunder i at webudvikleren skal bruge semantiske tags, som er relevante, så siden er lettere at forstå og finde.
Niels Østergaard "intro til html.pdf"

Kodeskik Med god kodeskik menes der at webudvikleren gør at koden er letlæselig. For at gøre koden letlæselig, er det vigtigt at det hele ikke står ud i en lang smører, men at koden er delt op i afsnit og linjer. Derudover kan webudvikleren også tilføje kommentarer i koden, som gør det lettere at finde det man leder efter i koden.
Niels Østergaard "intro til html.pdf"

CSS

CSS er ligesom HTML et kodesprog, som styler hjemmesiden. CSS bestemmer udseendet på hjemmesiden, hvor HTML bestemmer indholdet. Men en CSS kode kan man fx vælge baggrundsfarvbe, tekstfarve osv. Når man starter et CSS eller HTML dokument slutter man navnet på filen med: .css eller .html Dette gør at browseren ved hvilken fil der er tale om.
Eksempler på CSS koder:
Background-color: Red; = Baggrundsfarven
